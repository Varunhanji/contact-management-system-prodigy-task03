# 📇 Contact Management System

This project is a simple **Contact Management System** developed as part of **ProDigy InfoTech Internship - Task 03**.

---

## 📌 Project Description

The Contact Management System allows users to store, manage, and organize contact information efficiently.

Users can:
- Add new contacts
- View all saved contacts
- Edit existing contact details
- Delete contacts

The contacts can be stored either in memory or in a file for persistent storage.

---

## 🚀 Features

- ➕ Add new contacts (Name, Phone, Email)
- 📋 View contact list
- ✏️ Edit existing contacts
- ❌ Delete contacts
- 💾 File-based storage (optional for persistence)
- 🖥️ Simple CLI-based interface

---

## 🧠 How It Works

1. The program displays a menu with options.
2. User selects an action:
   - Add Contact
   - View Contacts
   - Edit Contact
   - Delete Contact
   - Exit
3. Based on the choice, the program performs the operation.
4. Data is stored and updated accordingly.
